# Tasmania Motorcycle Practice DKT

A free, mobile-friendly practice test for the Tasmanian Motorcycle Driver Knowledge Test.

## How to use

1. Open `index.html` in any web browser
2. Or host it on GitHub Pages, Netlify, or any static hosting

## Adding images

Place diagram images in the `/images/` folder:
- Intersection diagrams: `q1.png`, `q14.png`, etc. (800x500px)
- Road signs: `q5.png`, `q89.png`, etc. (400x400px)
- Motorcycle questions: `qm7.png`, `qm10.png` (800x500px)

The app will show a text fallback for any missing images.

## Test history

Past attempts are saved in your browser's local storage. No account needed.
